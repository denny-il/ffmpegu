# Overview

`ffmpegu` is a TypeScript-first FFmpeg command builder for backend runtimes. It provides:

- Fluent command construction (`ffmpegu.command`)
- Typed inputs/outputs (`ffmpegu.input`, `ffmpegu.output`)
- Options helpers (`ffmpegu.options.*`)
- Filters and filtergraphs (`ffmpegu.filters.*`)
- FFprobe integration (`ffmpegu.probe`, `ffmpegu.createFFprobeRunner`)
